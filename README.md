# linux
linux study and app

2006 年开始接触 Unix的分支 freeBSD , 坚如磐石的服务器系统。
     好用，但不适合个人电脑，软件偏少
2007 找到了 LFS(linux from scratch) 以某个linux系统为平台，全新配置和编译一个自己定制的linux系统，从lib到core到usr，再到Xwindow。
     编译出来的 lfs 不到200M，使用了半年多。缺少包管理工具，更新就要重新编译。
     此时，ubuntu 兴起了，从此开始了ubuntu + vm的双系统之旅， 直到2013年。
2014 在新公司使用 centos 作为服务器，至今(2016-11)

lubuntu VS linuxMint 为低配的旧电脑找合适的系统。
